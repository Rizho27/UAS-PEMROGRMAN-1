# Sistem Reservasi Hotel

Project UAS Mata Kuliah **Pemrograman 1** menggunakan **Java NetBeans** dan **MySQL**.

## Deskripsi

Aplikasi ini dibuat untuk membantu proses reservasi hotel, mulai dari pengelolaan data kamar, data tamu, reservasi, pembayaran, hingga menampilkan laporan data pembayaran.

---

## Fitur

- Login
- Dashboard
- CRUD Data Kamar
- CRUD Data Tamu
- CRUD Data Reservasi
- CRUD Data Pembayaran
- Laporan Pembayaran
- Logout

---

## Tools yang Digunakan

- Java
- Apache NetBeans IDE
- MySQL
- phpMyAdmin
- JDBC Driver MySQL

## 💾 Database

Import file berikut ke phpMyAdmin

```
database/hotel_reservasi.sql
```

---

## Cara Menjalankan Project

1. Clone atau download repository ini.
2. Import database `hotel_reservasi.sql` ke MySQL/phpMyAdmin.
3. Buka project menggunakan Apache NetBeans.
4. Pastikan konfigurasi database pada file `Koneksi.java` sudah sesuai.
5. Jalankan file `Main.java`.
